# Benvenuto sul mio Blog GitHub Pages

Come primo passaggio abbiamo abilitato le pagine GitHub.

Sotto il nome del repository, cliccando su Impostazioni,
nella sezione "Pagine GitHub", dall'elenco a discesa Sorgente,
selezioniamo il ramo principale Master.
GitHub prende il contenuto che si trova sul ramo master e pubblica un sito Web in base al suo contenuto.

Nel secondo passaggio personalizziamo la Home Page.

Per personalizzare la Home Page possiamo aggiungere contenuto al file index o al file README.md.
GitHub Pages cerca prima il file index e se non lo trova utilizza il contenuto del file README.md per creare la home page.
